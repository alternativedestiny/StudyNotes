## 2. Vue安装方法

1. 本地安装方法

    安装vue，在项目目录下的cmd

    ```cmd
    npm init -y
    npm i vue
    ```

    在前端页面引入vue

    ```html
    <script src="/node_modules/vue/dist/vue.js"></script>
    ```

2. 在线CDN方法

    ```html
    <script src="https://cdn.jsdelivr.net/npm/vue"></script>
    ```
